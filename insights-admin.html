⚙️ Hướng Dẫn Thiết Lập SharePoint List

Tạo một SharePoint List tên InsightsResponses với các cột sau:

  • Title (Single line) — Họ tên nhân viên (cột mặc định)
  • EmployeeRole (Single line) — Chức danh
  • Department (Single line) — Phòng ban
  • Answers (Multiple lines) — JSON câu trả lời
  • RedScore (Number) — Điểm Red (0–24)
  • YellowScore (Number) — Điểm Yellow
  • GreenScore (Number) — Điểm Green
  • BlueScore (Number) — Điểm Blue
  • PrimaryColor (Single line) — Màu chính
  • SecondaryColor (Single line) — Màu phụ

Sau đó upload cả 2 file HTML vào thư viện tài liệu SharePoint. Mở file này (insights-admin.html) — hệ thống sẽ tự detect site URL.
